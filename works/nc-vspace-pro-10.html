In February 2015, ZeroDesktop acquired core technology assets from NComputing. NComputing is a desktop virtualization company with more than 70,000 customers in 140 countries.

I worked on redesigning NComputing's Windows software family, vSpace Pro. The original UI was about a decade old, and consist of basic Windows UI elements. Also, overall UI flow was very complicated and hard to understand. Other project managers and I brainstormed to come up with ideas how we can turn this complicated UI to into more intuitive one.

I created wireframe using PowerPoint and shared with the development team to get feedback before I started the actual visual design.

vSpace Pro 10 is consist of vSpace Console and vSpace Manager.

vSpace Console is for managing the host computers and connected thin client devices.

vSpace Manager is for allocating software licenses and Premium Features to each host computer.
